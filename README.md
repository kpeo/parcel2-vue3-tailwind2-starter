# parcel2-vue3-tailwind2 starter

Basic initial Vue3 project with Parcel2 and Tailwind2

## Project setup
```
yarn install
# or
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
# or
npm run serve
```

### Compiles and minifies for production
```
yarn build
# or
npm run build
```

### Lints and fixes files
```
yarn lint
# or
npm run lint
```
